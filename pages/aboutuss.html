About Me

Hello, I’m Agustín Matías Gómez, a passionate Fullstack Developer. This project was a final delivery for my Coder House course, where I was tasked with creating an e-commerce platform. I chose to build an e-commerce site dedicated to CrossFit products because this discipline inspires me, and I wanted to create something meaningful that resonates with my passion.

This project took me about seven months to complete, as I wanted to ensure it provided both a great user experience and robust functionality. My goal with this project is to showcase my versatility and adaptability as a developer, demonstrating how I can create a seamless experience for users while building scalable and maintainable solutions.

Technologies & Tools Used

  • HTML, CSS, JavaScript
  • Node.js & Express.js
  • MongoDB & Firebase
  • Git & GitHub
  • AWS S3 for image storage
  • Responsive Web Design (RWD)
  • REST APIs & Fetch
